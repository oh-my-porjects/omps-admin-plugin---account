# 账号管理 模块事实

## 职责

账号公共模块 — 平台官方维护的基础设施模块，提供后台账号、登录、session、临时超管账号机制等能力。

## 数据表

- `account_accounts`
- `account_role_bindings`

## API

- `POST /api/account/login`：后台账号登录
- `GET /api/account/me`：查询当前后台账号
- `POST /api/account/_validate-password`：校验账号密码
- `POST /api/account/create`：创建后台账号
- `GET /api/account/list`：查询后台账号列表
- `GET /api/account/detail`：查询后台账号详情
- `PUT /api/account/update`：更新后台账号
- `PUT /api/account/reset-password`：重置后台账号密码
- `POST /api/admin/account/delete`：删除后台账号
- `POST /api/account/check-permission`：检查后台权限
- `GET /api/account/hello`：查询账号模块元信息
- `POST /api/account/admin/ping`：后台账号管理接口探活
- `POST /_internal/method-call/admin_account`：内部方法调用；`CreateTemporaryAdmin` 生成临时超级管理员账号，只能经 runtime 管理面触发
- `POST /_internal/scheduled-trigger/admin_account`：内部定时任务触发
- `POST /_internal/selftest/admin_account`：内部自测入口

## 实时事件

用户前端事件：

- 当前模块没有自有用户前端推送事件。

后台提醒事件：

- 当前模块没有自有后台实时提醒事件。

## 测试资产

- `tests/data.yaml` 声明本模块测试涉及的数据表、种子、清理和快照范围。
- 每个 `api-docs/endpoints/*.json` 里的 `endpoint_key` 都应有对应 `tests/*.test.json`。
- 新增、删除或修改接口时，必须同时更新 `api-docs/`、测试文件和必要的数据种子。
